## How Sync Works

### Offline-First Design
ScriptMine works perfectly offline. All your changes are saved locally and synced when you're online.

### Automatic Sync
When enabled, ScriptMine syncs:
- Every X minutes (configurable)
- When you open the app
- When network becomes available
- After making changes (if online)

### Conflict Resolution
If you edit the same script on multiple devices:
1. ScriptMine detects the conflict
2. Keeps the most recent version
3. Notifies you if manual review is needed

## Troubleshooting

### Sync Not Working
1. Check internet connection
2. Verify you're signed in
3. Try manual sync
4. Check Settings > Sync Settings

### Can't Sign In
1. Check internet connection
2. Verify Google account is active
3. Try again in a few minutes
4. Contact support if persists

### Data Not Syncing
1. Enable automatic sync in Settings
2. Check WiFi-only setting
3. Trigger manual sync
4. Reset sync if needed

### App Running Slow
1. Clear cache in Settings
2. Close and reopen app
3. Check available storage
4. Update to latest version

## FAQ

**Q: Do I need to sign in?**
A: No, you can use ScriptMine completely offline. Sign in only if you want cloud sync.

**Q: Is my data safe?**
A: Yes! Data is encrypted on device and in transit. Only you can access your scripts.

**Q: How much data does sync use?**
A: Very little! We use compression and incremental sync. Enable "WiFi Only" to be safe.

**Q: Can I use multiple devices?**
A: Yes! Sign in with the same Google account on all devices.

**Q: What happens if I delete the app?**
A: If signed in, your data is safe in the cloud. Reinstall and sign in to restore.

**Q: How do I backup my data?**
A: Use Settings > Data Management > Export Data to create a backup file.
